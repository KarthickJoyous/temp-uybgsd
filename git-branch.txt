1. Local branch

git branch

2. Remote branch

git branch -r

3. Local + Remote branch

git branch -a

4. Create branch

git branch v1

5. Switch to branch

git checkout v1
git switch v1

6. Create branch & switch 

git switch -c v2

7. Rename branch :

git branch -m v1.1

8. Delete branch (local) :

git branch -d v1

9. Delete branch (remote) :

git branch -d v1

git push origin --delete v1
